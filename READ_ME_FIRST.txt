First of all, run POMConverter.exe
Once the application is opened, put your song data in the text area and press "Convert" 
to export your song to an mp3 file.  The file will be saved in the exportedSongs folder.
Be patient the conversion can take up to a couple minutes.

If you get an error similar to this:
To run this application, you must install one of the following versions of the .NET framework: v2.0.50727
Go to this link and install the latest .net framework. (I'm not responsible for any problem you may have doing this)
http://msdn2.microsoft.com/en-us/netframework/aa569263.aspx

The application will not work on unix and mac operating systems either.
